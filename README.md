# HTML to Image Converter

A simple, production-ready web application that converts HTML/CSS code into PNG images using the HTML/CSS to Image API.

## Features

- ✅ Pure HTML/CSS/JS frontend (no frameworks)
- ✅ Secure Node.js backend (API key never exposed)
- ✅ Clean, responsive UI
- ✅ Loading states and error handling
- ✅ Download generated images
- ✅ Custom width/height settings
